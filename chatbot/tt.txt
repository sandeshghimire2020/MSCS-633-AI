you will create a terminal client that allows you to chat with a bot using Django/Python and Chatterbot. 

ChatterBot is a machine-learning based conversational dialog engine build in Python which makes it possible to generate responses based on collections of known conversations.  

 
Input/Output: 
A typical input/output would be something like this: 

user: Good morning! How are you doing?
bot: I am doing very well, thank you for asking.
user: You're welcome.
bot: Do you like hats?
